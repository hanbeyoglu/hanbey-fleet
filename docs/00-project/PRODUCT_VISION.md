# Executive Summary

Hanbey Fleet is a modern fleet operations platform designed for taxi owners and small fleet businesses.

Instead of manually tracking income, expenses, HGS tolls, maintenance records and driver reports using Excel and WhatsApp, Hanbey Fleet centralizes all operational data into a single platform.

The system focuses on reducing manual work, preventing financial mistakes, and providing complete operational visibility.

---

# Vision

To become the simplest and most intelligent fleet management platform for small and medium-sized fleet owners.

Hanbey Fleet should allow owners to spend less time managing operations and more time making business decisions.

---

# Mission

The system should calculate.

The user should decide.

Hanbey Fleet automates operational processes so users never need spreadsheets for daily vehicle management.

# Problems We Solve

Current problems include:

- Manual Excel bookkeeping
- Manual HGS verification
- Driver payment mistakes
- Missing maintenance history
- Lost invoices
- Difficulty tracking vehicle expenses
- No centralized operational history
- No real-time dashboard

---

# Product Principles

## Automation First

If something can be automated, it should never require manual work.

---

## Simplicity First

The application should remain understandable within five minutes.

---

## Mobile First

Every common operation must be executable from a mobile device.

---

## Timeline First

Every operational activity becomes part of the vehicle timeline.

---

## AI Ready

The architecture must support future AI integrations without structural changes.

---

## Single Source of Truth

Every piece of information has exactly one owner.

Example:

- HGS → Bank API
- Driver Report → Driver
- Maintenance → Expense + Maintenance Detail

# Success Metrics

The owner should instantly answer:

- Who currently has the vehicle?
- Today's revenue?
- Today's HGS?
- Monthly profit?
- Last maintenance?
- Current vehicle expenses?
- Driver performance?
- Vehicle profitability?

---

# Out of Scope (MVP)

The first version will NOT include:

- Accounting integrations
- E-Invoice
- Mobile apps
- Multi-company support
- AI Assistant
- GPS Tracking

These features are planned for future releases.
