# Hanbey Fleet

Version: 1.0.0

---

# Project Charter

## Vision

Hanbey Fleet is a modern fleet operations platform designed for taxi owners and small fleet operators.

The goal is not only to manage vehicles but to simplify daily operations by automating repetitive tasks, reducing manual bookkeeping, and providing a single source of truth for every vehicle.

The platform should evolve into an intelligent operations assistant rather than a traditional accounting software.

---

# Mission

Our mission is simple:

> The system should do the calculations.
> The user should only make decisions.

The owner should never need Excel to understand the current state of the fleet.

---

# Product Philosophy

Hanbey Fleet is **NOT**

- an accounting software
- an ERP
- an HGS application

Hanbey Fleet **IS**

a Fleet Operations Platform.

Everything revolves around the operational lifecycle of a vehicle.

---

# Core Principles

## 1. Automation First

Whenever data can be collected automatically,
manual input must not exist.

Examples

- HGS → API
- WhatsApp → Parser
- OCR → Expense

---

## 2. Mobile First

Every daily operation must be executable from a mobile device.

---

## 3. Three Click Rule

No common operation should require more than three clicks.

---

## 4. Single Source of Truth

Every piece of information has exactly one owner.

Example

HGS

Owner:

İş Bankası API

never manual.

---

## 5. Event Driven

Everything that happens inside the system becomes an Event.

Examples

Vehicle Assigned

Fuel Added

Expense Created

Maintenance Completed

HGS Imported

Settlement Approved

Vehicle Returned

---

## 6. Timeline First

Every vehicle has a timeline.

The timeline represents the complete operational history of that vehicle.

---

## 7. AI Ready

The system must be designed so AI can be integrated without changing the architecture.

---

## 8. User Experience over Features

Simple beats powerful.

The owner should be able to understand the entire system within five minutes.

# Success Criteria

The owner should be able to answer the following questions instantly:

Who currently has the vehicle?

How much money came in today?

How much HGS was spent today?

Did the driver report the correct amount?

When was the brake replaced?

How much has this vehicle cost this year?

Which driver generated the most revenue?

What is today's profit?

# Long Term Goal

Hanbey Fleet should become a SaaS platform capable of managing multiple companies, vehicles, drivers and operational workflows.
